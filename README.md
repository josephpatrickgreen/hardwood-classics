# hardwood-classics

Chain Net prototype scaffolding for a pick-a-path basketball roguelike in Unity.

## Project Layout

The repository includes the requested Unity folder structure and modular C# systems for:

- ScriptableObject data definitions (characters, teams, trinkets, specials, courts, nodes, events)
- Runtime run/team/player/map state
- Match/map flow managers
- Arcade camera, hype/heat/foul/dirty/fight systems
- Save/unlock foundations

## Notes

- This is a vertical-slice-ready code skeleton intended to be opened in Unity and wired to prefabs/UI.
- `SampleDataLibrary` helps verify phase-1 minimum data counts.
